CherryMusic Control (Firefox addon)
===============

Easier media player controls for [CherryMusic](http://www.fomori.org/cherrymusic/)

###Available buttons
* Play/Pause
* Previous track
* Next track
* Media info
* Logout

###Hotkeys
* Play/Pause - "Ctrl + Space"
